Write-up notes

Type y to begin the challenge. 

1) When run without any options, nmap performs a TCP port scan of the top
   1000 ports. Run a default nmap scan of 127.0.12.25 and see which port is open.

  ────────────────────────────────────────────────────────────────────────────────────────

elfe3A0889583b3c:~ $
Write-up notes

Enter nmap 127.0.12.25 to perform a default scan of the target host. 

2) Sometimes the top 1000 ports are not enough. Run an nmap scan of all TCP
   ports on 127.0.12.25 and see which port is open.

  ────────────────────────────────────────────────────────────────────────────────────────

elfe3A0889583b3c:~ $ nmap 127.0.12.25
Starting Nmap 7.80 ( https://nmap.org ) at 2025-12-09 20:59 UTC

elfe3A0889583b3c:~ $
Write-up notes

With the default scan we discovered port 8080 is opn, lets use nmap 127.0.12.25 -p- to scan all TCP ports. 

3) Nmap can also scan a range of IP addresses.  Scan the range
   127.0.12.20 - 127.0.12.28 and see which has a port open.

  ────────────────────────────────────────────────────────────────────────────────────────

Write-up notes

To scan the IP range 127.0.12.20 - 127.0.12.28, use the command nmap 127.0.12.20-28.

To determine the service running on port 8080, use the command nmap 127.0.12.25 -p 8080 -sV, the "-sV" flag enables version detection.
